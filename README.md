# Flower Shop Drawings

Bouquet drawings for Yang's Flower Shop — a weekly Monday newsletter.

Each PNG/JPG is one week's bouquet. Filename = flower name (e.g. `magnolia.png`, `peony.png`).

The newsletter agent picks the right drawing each Monday and embeds it in the email via the raw URL:

`https://raw.githubusercontent.com/dolh77/flower-shop-drawings/main/<filename>`
